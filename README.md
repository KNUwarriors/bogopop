영화 리뷰 웹사이트: 보고팝!(BOGOPOP!)
=================

### 1. 프로젝트 소개
우리들만의 영화 리뷰 사이트를 만들어보자!
영화를 본 후 팝콘으로 내 점수를 표현한다는 의미의 보고POP
